## The Non-Negotiable 5 (Zero Equipment, 20 minutes)

When time or space is severely limited, this sequence covers all major muscle groups:

```yaml
exercise_1:
  name: Push-Up (or Incline on bed)
  sets_reps: "4 × max reps (to 2 RIR)"
  focus: chest, shoulders, triceps
exercise_2:
  name: Reverse Lunge
  sets_reps: "3 × 12 per side"
  focus: quads, glutes, hamstrings
exercise_3:
  name: Pike Push-Up
  sets_reps: "3 × 10"
  focus: shoulders
exercise_4:
  name: Glute Bridge
  sets_reps: "3 × 20 (2s hold at top)"
  focus: glutes, hamstrings
exercise_5:
  name: Plank + Side Plank
  sets_reps: "3 × 45s each"
  focus: core stability
total_time: ~20 minutes
```

## Full Hotel Room Workout (40 Minutes)

### Warm-Up (5 min)
- Arm circles, leg swings, hip circles — 10 reps each
- Bodyweight squat × 15
- Inchworm × 5

### Block A — Lower Body
| Exercise | Sets × Reps | Notes |
|---|---|---|
| Bodyweight Squat | 3 × 20 | Slow eccentric (3s) |
| Reverse Lunge | 3 × 12/side | |
| Single-Leg RDL (bodyweight) | 3 × 10/side | Hinge pattern |
| Glute Bridge | 3 × 20 (2s hold) | |

### Block B — Upper Body
| Exercise | Sets × Reps | Notes |
|---|---|---|
| Push-Up (standard or incline on bed) | 4 × max to 2 RIR | |
| Pike Push-Up | 3 × 10 | Shoulders |
| Diamond Push-Up | 3 × 10 | Triceps |
| Band Row (if band available) OR Door-Frame Row | 3 × 15 | |

### Block C — Core
| Exercise | Sets × Reps | Notes |
|---|---|---|
| Plank | 3 × 45s | |
| Dead Bug | 3 × 10/side | |
| Side Plank | 2 × 30s/side | |

## Hotel Gym Equipment Map

If the hotel has a small gym, use this priority order:

```yaml
equipment_found: treadmill
use_for: "10–20 min walking/jogging warm-up or cardio finisher"

equipment_found: dumbbells_only
use_for: [chest_004, back_010, legs_002, shoulders_002, arms_002]
best_exercises: "Goblet squat, DB RDL, DB press, DB row, DB shoulder press"

equipment_found: cable_machine
add: [back_007, back_008, arms_009_cable, shoulders_005]

equipment_found: pull_up_bar
add: [back_005, back_006, core_010]
```

## Progression Without Equipment

```yaml
push_up_progression:
  week_1: "Incline push-up on bed"
  week_2: "Standard push-up"
  week_3: "Slow eccentric (3s down)"
  week_4: "Decline push-up (feet on bed)"
  week_5: "Archer push-up"

squat_progression:
  week_1: "Bodyweight squat × 20"
  week_2: "Slow tempo squat (3s down)"
  week_3: "Single-leg squat to chair"
  week_4: "Pistol squat (assisted)"

pulling_without_bar:
  option_1: "Door-frame row: hold door frame, lean back, row yourself in"
  option_2: "Resistance band row anchored at door"
  option_3: "Backpack row: fill backpack with books for load"
```

## Low-Noise Options (Early Morning / Late Night)
```yaml
exercises_suitable:
  - Plank and variations (core_001, core_009)
  - Dead Bug (core_002)
  - Glute Bridge (legs_013)
  - Wall Sit (legs_015)
  - Slow-tempo push-ups (no impact)
  - Resistance band exercises
  - Stretching and mobility

avoid_at_quiet_hours:
  - Jump squats
  - Burpees
  - Any jumping or running in place
```

```yaml
weight: "~200g; fits in any bag"

  - arms_004  # Bicep curl
  - arms_010  # Tricep extension
  - back_012  # Band row
  - chest_012 # Band fly
  - shoulders_008 # Band pull-apart

```

## Sample 5-Day Travel Program

```yaml
monday: "Full body (40-min session above)"
tuesday: "20-min walk/jog + core focus (plank series + dead bug)"
wednesday: "Push focus: push-up variations + pike push-up + tricep"
thursday: "Active rest: walk 8,000 steps, stretch"
friday: "Pull + legs: band rows + reverse lunge + glute bridge + single-leg RDL"
saturday: "Non-negotiable 5 (20 min)"
sunday: "Rest or walk"
```
